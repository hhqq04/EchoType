# EchoType
一个轻量级的 Windows 自动键盘输入模拟器，基于 C 语言和 Win32 API 编写。支持中文输入，通过模拟底层键盘事件，将粘贴的文本自动输入到任何目标窗口中。

1. 🖥️ 远程桌面与虚拟机 (Remote Desktop & VMs)
这是本工具最强的应用场景。

场景描述：当你使用 RDP (远程桌面)、TeamViewer、AnyDesk，或者在 VMware/VirtualBox 虚拟机中操作时，宿主机和虚拟机之间的剪贴板共享经常失效，导致无法复制文字进去。
解决方案：在宿主机运行本工具，将文本“打”进虚拟机里，完美绕过剪贴板限制。
2. 🚫 突破“禁止粘贴”的限制 (Bypass Anti-Paste)
场景描述：某些网银控件、安全登录框、或者在线考试系统，为了安全或防止作弊，禁用了右键粘贴和 Ctrl+V 功能。
解决方案：本工具模拟的是物理键盘输入，网页和软件无法区分是人手敲的还是程序打的，因此可以强制输入长密码、身份证号或代码。
3. 🎮 游戏指令与自动喊话 (Gaming & Consoles)
场景描述：很多全屏游戏（如 Minecraft, CS:GO, GTA）的控制台（Console）或聊天窗口不支持外部粘贴。
解决方案：提前录入好复杂的作弊码、服务器指令或喊话内容，一键自动输入。
4. 💻 服务器运维 (Server Maintenance)
场景描述：通过 VNC 或某些老旧的 KVM 交换机管理服务器时，终端窗口（Terminal）往往不支持粘贴长指令。
解决方案：使用本工具将长段的 Shell 脚本或配置命令精准地输入到终端中。
5. 🧪 自动化测试 (QA & Automation Testing)
场景描述：测试人员需要测试软件输入框对“特殊字符”、“超长文本”或“中文GBK/Unicode编码”的兼容性。
解决方案：通过本工具可以稳定、重复地进行压力测试，模拟用户极速输入的情况。
6. 📝 传统软件数据录入 (Legacy Software)
场景描述：某些古老的 ERP 系统、进销存软件（基于 DOS 或旧版 Windows），完全不支持现代的剪贴板操作。
解决方案：代替人工进行枯燥的重复性文字录入工作。
